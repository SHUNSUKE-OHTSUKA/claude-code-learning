## メモ（実施日・気づき）

- Claudeの責務分離機能であるWorktreeとSubAgentを学習した
- Worktreeはブランチ単位でAgentを権限やファイル分離するなど厳密に分離したいときに利用するもの（作業ディレクトリ（環境）を切り分ける機能）で、SubAgentはWorktreeほどではないが軽度な分離をしたいときに利用するもの（AIの役割（権限・行動）を切り分ける機能）であった
- Worktreeの使いどことろ…大規模な開発/リファクタ、別個機能の並列開発、同一仕様の複数実装
- SubAgentの使いどころ…軽い調査タスクを並列でする、並列テスト
- /loopはcron的な設定ができる
- セッションスコープなので/exitしたら設定内容は破棄される
