# 進捗トラッカー

`ROADMAP.md` の各Phaseの進捗をここでチェックする。チェックを入れたらコミットし、下の学習ログに一言残す。

## チェックリスト

### Phase 0｜準備
- [ ] CLIインストール・認証完了
- [ ] 初回起動・簡単な質問を試した

### Phase 1｜基礎操作
- [ ] CLAUDE.mdを作成した
- [ ] Grep/Globだけでファイル探索させた
- [ ] バグ修正を1件完結させた
- [ ] commitメッセージの品質を確認した

### Phase 2｜効率化
- [ ] settings.jsonに許可リストを設定した
- [ ] Plan Modeを使って計画レビューをした
- [ ] カスタムスラッシュコマンドを1つ作った

### Phase 3｜拡張
- [ ] SKILL.mdを1つ作った
- [ ] MCPサーバーを1つ接続した
- [ ] hooksでformatter自動実行を設定した

### Phase 4｜自動化・並列化
- [ ] 並列調査を依頼した
- [ ] git worktreeでリスクの高い変更を隔離した
- [ ] 定期実行タスクを1つ組んだ

### Phase 5｜熟達
- [ ] チームとCLAUDE.mdをレビューし合った
- [ ] `/code-review` → `/security-review` の流れを実践した
- [ ] Claude Agent SDKに目を通した

## 学習ログ

日付・やったこと・気づきを1行ずつ追記していく。

```
2026-07-15  リポジトリを作成し、ロードマップと教材を整備した
```
