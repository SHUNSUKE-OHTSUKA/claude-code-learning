# 進捗トラッカー

## チェックリスト

### Phase 0｜準備

- [x] CLIインストール・認証完了

- [x] CLAUDE.mdを作成した

### Phase 3｜拡張

### Phase 4｜自動化・並列化

- [ ] 並列調査を依頼した

- [ ] チームとCLAUDE.mdをレビューし合った
- [ ] `/code-review` → `/security-review` の流れを実践した
